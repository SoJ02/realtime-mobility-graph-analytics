# DPS Project: Graph Analytics Pipeline (Neo4j + Kafka + Kubernetes)

This project demonstrates a distributed graph analytics pipeline using NYC taxi trip data.

## Structure

- `Phase 1/` - static graph loading and algorithm execution with Neo4j
- `Phase 2/` - streaming pipeline with Kafka and Kubernetes into Neo4j

## Key Capabilities

- Real-time ingestion pipeline for trip events
- Graph construction and updates in Neo4j
- Graph analytics workflows including BFS and PageRank

## Tech Stack

- Python
- Neo4j
- Apache Kafka
- Docker
- Kubernetes (Minikube)
